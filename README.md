Install
---------

``` bash
npm install w-ui
```

Quick Start
---------

``` javascript
import Vue from 'vue'
import {
  Select,
  Button
  // ...
} from 'w-ui'

Vue.component(Select.name, Select)
Vue.component(Button.name, Button)
```

For more information, please refer to Quick Start in our documentation.
